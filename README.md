# SMS-Bomber

Описание настроек:
- Номер цели
  - Мобильный номер телефона для спама, указывать без символов
- Кол-во циклов
  - Количество циклов которые выполнит программа, в одном цикле 47 сайтов


Для Termux:
- pkg install python
- pip install requests
- git clone https://github.com/Ikysu/SMS-Bomber/bomber.py
- python bomber.py
